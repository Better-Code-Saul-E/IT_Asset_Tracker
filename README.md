# IT_Asset_Tracker
A command-line tool for tracking IT assets, built with Python and SQLite. This tool allows you to add, view, update, delete, and export asset information

## Features
**Database Setup**
- SQLite database for storing asset information

**Add & List Assets**
- Add new assets via commands
- List all assets in a neat table using the tabulate library
![Deleting an Asset](images/DisplayData.png)

**Update & Delete Assets**
- Modify and delete asset information with confirmation to avoid accidents
![Deleting an Asset](images/DeleteData.png)

**Exporting Assets**
- Export asset data to CSV and JSON formats for reporting
![Deleting an Asset](images/ExportData.png)
